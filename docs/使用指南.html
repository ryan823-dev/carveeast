🔌 API 接口文档

Stripe 支付 API

POST /api/stripe/create-checkout

创建 Stripe Checkout Session 进行课程购买。

POST /api/stripe/webhook

处理 Stripe Webhook 回调，验证支付状态。

PayPal 支付 API

POST /api/paypal/create-order

创建 PayPal 订单。

POST /api/paypal/capture-order

捕获已批准的 PayPal 订单。

OSS 上传 API

POST /api/oss/get-signature

获取 OSS 上传签名（用于客户端直传）。

其他 API

GET /api/courses
GET /api/works
GET /api/artists

🚀 部署说明

Vercel 部署步骤

连接 GitHub

在 Vercel Dashboard 中导入 GitHub 仓库。

配置环境变量

在 Vercel 项目 Settings → Environment Variables 中添加所有凭证。

部署项目

点击 Deploy 开始部署，Vercel 会自动构建并发布。

配置域名

在 Settings → Domains 中绑定自定义域名。

生产环境配置

✅
切换到生产模式

将 PAYPAL_MODE 从 sandbox 改为 live，并配置真实的 Stripe 密钥。

视频上传流程

处理视频

使用 video_processor.py 提取帧、生成 GIF、优化分辨率。

上传到 OSS

使用 oss_upload.py 自动上传视频和缩略图到阿里云 OSS。

更新配置

将生成的视频 URL 更新到 src/lib/course-videos.ts。
