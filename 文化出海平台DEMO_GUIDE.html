📖 平台简介

CarveEast 是一个专业的当代中国艺术收藏平台，专注于篆刻、书法、水墨画、刻瓷等传统艺术形式。 平台为收藏家和艺术家搭建专业桥梁，提供作品展示、在线收藏、拍卖参与等全方位服务。

Next.js 16 TypeScript Tailwind CSS Prisma SQLite/PostgreSQL Vercel

🚀 使用流程

1

注册/登录

使用演示账号登录平台，选择艺术家或收藏家身份

2

完善资料

艺术家完善个人简介、上传代表作品和工作室照片

3

浏览/上传

收藏家浏览作品，艺术家上传新作品到平台

4

互动/管理

收藏家收藏作品、参与拍卖，艺术家管理作品和查看数据

⚙️ 技术特点

💡 技术架构

  • 前端框架：Next.js 16 + TypeScript + Tailwind CSS
  • 数据库：SQLite（开发）/ PostgreSQL（生产）
  • 部署平台：Vercel（全球 CDN 加速）
  • 图片 CDN：Unsplash（免费商用）
  • 认证系统：基于 localStorage 的演示认证

✅ 已完成功能

  • ✅ 数据库连接和 CRUD 操作
  • ✅ 艺术家后台管理系统
  • ✅ 作品上传和管理功能
  • ✅ 真实图片和头像展示
  • ✅ Vercel 生产环境部署
  • ✅ 响应式设计（手机/平板/桌面）

⚠️ 演示环境说明

📌 重要提示

  • 数据持久化：演示环境使用 SQLite，数据保存在本地。生产环境建议使用 Vercel Postgres（免费额度）
  • 图片资源：所有图片来自 Unsplash（免费商用图库）
  • 支付功能：演示环境未接入真实支付系统
  • 域名绑定：当前使用 Vercel 提供的临时域名，可随时绑定自定义域名
