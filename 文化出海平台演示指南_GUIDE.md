## 🎨 平台简介

CarveEast 是一个专业的当代中国艺术收藏平台，专注于：
- 篆刻艺术
- 书法
- 水墨画
- 刻瓷艺术
- 版画
- 油画

平台提供艺术家展示、作品收藏、拍卖参与等功能，为收藏家和艺术家搭建专业桥梁。

## 🎯 使用流程

### 收藏家流程

1. **浏览作品** → 按兴趣浏览或搜索
2. **查看详情** → 了解作品信息和文化背景
3. **收藏作品** → 添加到收藏夹
4. **询价/购买** → 提交询价或直接购买
5. **完成订单** → 支付和物流安排

### 艺术家流程

1. **注册账号** → 提交艺术家资料
2. **完善资料** → 上传头像、简介、代表作品
3. **上传作品** → 添加新作品到平台
4. **管理作品** → 更新状态、编辑信息
5. **查看数据** → 关注浏览量和收藏数
