## 🌐 网站资源优化方案

### 视频使用策略

#### 问题：视频文件太大（20-900MB）

#### 解决方案：

**1. 网站展示层（轻量）**
```
原始视频 → 截图/GIF → 网站展示
            ↓
         30 秒预览 → 嵌入播放器
```

**实施：**
- 每个视频提取 3-5 张关键帧
- 制作 10-15 秒 GIF 动图
- 上传 720p 预览版（<50MB）到 Vimeo/B 站
- 网站嵌入播放器

**2. 付费课程层（重量）**
```
原始视频 → 阿里云盘/Vimeo Pro → 付费用户下载
```

**实施：**
- 原画质保存（1080p/4K）
- 使用第三方平台托管
- 付费后提供下载链接或观看权限

**3. 社交媒体层（引流）**
```
原始视频 → 剪辑 1-3 分钟 → TikTok/YouTube Shorts/Instagram
```

**实施：**
- 提取精彩片段
- 添加英文字幕
- 竖屏适配移动端

### 图片使用策略

**1. 作品图片**
- 原图保存（用于产品详情页放大）
- 生成 800x800 中等图（列表页）
- 生成 200x200 缩略图（画廊）
- WebP 格式转换（减小 50% 体积）

**2. 教学截图**
- 从视频中提取关键步骤
- 制作教学图解
- 添加英文标注
