🚀 CarveEast 部署指南

5 分钟部署你的当代中国艺术收藏平台

点击这里开始部署 →
1 访问 Vercel 并登录

点击上方按钮或访问 vercel.com

使用 GitHub 账号登录（推荐）或注册新账号

2 导入 GitHub 仓库

点击 "Add New Project" → "Import Git Repository"

选择仓库：ryan823-dev/carveeast

3 配置构建选项

Framework Preset: Next.js

Build Command: prisma generate && next build
Output Directory: .next
Install Command: npm install
4 添加环境变量

点击 "Environment Variables" → "Add New"

Key: DATABASE_URL
Value: file:./dev.db
⚠️ 重要：生产环境建议使用 PostgreSQL
可在 Vercel 项目中创建免费的 Vercel Postgres
5 点击部署

点击 "Deploy" 按钮，等待 2-5 分钟

部署成功后会获得一个 https://xxx.vercel.app 域名

🎉 部署完成！

你现在可以：

  • 分享网站链接给艺术家和合作伙伴
  • 访问艺术家后台管理作品
  • 随时绑定自定义域名

📦 数据库说明

当前配置使用 SQLite，适合演示和本地开发。

正式使用建议在 Vercel 创建 PostgreSQL 数据库（免费）：

1. 项目设置 → Storage → Create Database

2. 复制 CONNECTION_URL

3. 更新环境变量 DATABASE_URL
